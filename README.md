# 💱 Currency Exchange Management System

## 📖 Description

This project is a web application for managing currency exchange operations. It provides three main interfaces to facilitate exchange transactions using either real-time exchange rates or manually defined rates.
Features

### 1. Real-Time Exchange Interface
- Retrieves exchange rates from an external API.
- Displays up-to-date currency rates.
- Performs currency exchange transactions using live exchange rates.

### 2. Custom Exchange Rate Management
- Allows the administrator or responsible user to define custom exchange rates manually.
- Stores and manages the exchange rates in the database.
- Enables updating or modifying rates at any time.

### 💸 3. Transaction Interface
- Performs currency exchange transactions using the custom rates defined by the administrator.
- Automatically calculates the converted amount.
- Generates and prints a transaction receipt containing:
  - Transaction number
  - Source and target currencies
  - Exchange rate used
  - Amount exchanged
  - Date and time of the transaction

## Technologies Used

- **Frontend:** React.js
- **Backend:** Laravel
- **Database:** MySQL
- **API:** Exchange Rate API
- **HTTP Client:** Axios

## Main Functionalities

- Live exchange rate retrieval
- Custom exchange rate management
- Currency conversion
- Transaction history
- Receipt generation and printing
- User-friendly interface
